	
Here is some help about the 'replace' command :
 	
	Command : replace [entry:R] [name:R] [identifier:R] [password:R]
			Replaces an existing entry with a new one, can be used to modify the identifier or the password.
 
	Aliases : 'replace'
 
 
	Arguments qualifiers
		R = required.
		O = optional.
