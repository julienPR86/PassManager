 
Here is some help about the 'data' command :
 
	Description :
		Display the path to the actual data file if no arguments are provided. Otherwise, executes the sub-command specified.
 
	Format : data [action:O] [path:O]
 
	Arguments :
		action : The sub-command to execute. See 'Sub-commands' section for possibilities.
		path : Path to the new data file.
 
	Sub-commands :
		change : Changes the data file path to a new one.
 
	Aliases : 'data'
 
 
	Arguments qualifiers
		R = required.
		O = optional.
